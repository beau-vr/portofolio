Over mij

Wie ben ik?

Ik ben Benjamin Willem Vroegh een jongen met veel energie, en veel zin om nieuwe dingen uit te proberen en te leren. ben op de midelbareschool in aanrakring gekoem met programeren en daardoor ben ik daarna de opleding Webdeveloper gaan doen op het Grafisch Lyseum Utrecht. op dit moment zit ik daar in mijn tweede leerjaar, en ik heb het er heel erg naar mijn zin.

Wat kan ik

HTML

Dit is een beetje een standard taal maar ik gebruik het vaak om een basis van een website mee op te zetten. Ik vind deze taal niet zo moeilijk maar er kunnen nog wat dingen beter en i ben vol op bezig met me zelf er dus ook op te verbeteren.

Javascript

Dit is de taal waar ik het miste ervaring mee heb maar op dit moment het meeste plezier vandaan haal door dat het zo verschillend kan zijn want je kan het zowat overal voor gebruiken. Ik vind het ook heel leuk om er oude projecten mee op te knappen en te verbeteren. Dus ik probeer hier nog veel beter in te worden.

Photoshop

ik gebruik Photoshop als een manier van dingen in beeld te brengen hoe het er uit moet zien zodat ik al een idee heb wat het moet zijn en hoe ik dingen precies wil hebben zodat het er goed uit komt te zien.

Css

Voor dat ik er achter kwam hoeveel je kan doen met javascript vond ik css het boeiends door dat je heel makkelijk iets er kan laten uitzien zoals dat je zou willen. Dus ik heb hier al veel ervaring mee en weet hoe ik het goed kan toepassen.
